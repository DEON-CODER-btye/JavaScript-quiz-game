# Quiz Game

A simple Quiz Game built using HTML, CSS, and JavaScript.  
The application allows users to answer multiple-choice questions and view their final score at the end.

## Live Demo

🌐 [Play the Game](the-quiz-project.netlify.app)

## Features

- Multiple choice questions
- Instant answer selection
- Score calculation
- Simple and interactive UI
- Dynamic question loading

## How to Play

1. Start the quiz.
2. Read the question carefully.
3. Select the correct option.
4. Move to the next question.
5. At the end, your final score will be displayed.

## Technologies Used

- HTML
- CSS
- JavaScript (DOM Manipulation)

This project was created to practice JavaScript concepts such as:

- DOM manipulation

- Arrays and objects
- Game logic implementation
